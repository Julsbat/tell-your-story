# Setup
### Server starten:
```npm start``` (läuft auf Port 3333)

### Smartphone Testen
* Server und Handy müssen sich im selben WLAN befinden
* Server starten
* Am Handy **"server-ip":3333** mit dem Browser öffnen
* Unter Option **"Am Startbildschirm platzieren"** auswählen
